# 项目说明

AQChat 一个极速、便捷的匿名在线即时聊天室。

对标游戏后端开发，采用Netty作为通讯框架，支持高并发，高性能的即时通讯。全程无需HTTP协议，支持文本、图片、文件、音频、视频的发送和接收。

## 其他仓库

- 后端服务：<a href="https://github.com/zsdddz/AQChatServer">AQChat-Server</a>
